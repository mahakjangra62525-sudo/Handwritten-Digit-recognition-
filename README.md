# Handwritten Digit Recognition

A beginner-friendly Machine Learning project that recognizes handwritten digits (0–9) using the MNIST dataset.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Data preprocessing
- Model training
- Digit prediction
- Performance evaluation

## Dataset
MNIST Handwritten Digits Dataset
